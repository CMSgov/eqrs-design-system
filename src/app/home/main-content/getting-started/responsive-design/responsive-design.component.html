What is Responsive Design

Responsive web design (RWD) is a web development approach that creates dynamic changes to the appearance of a website, depending on the screen size and orientation of the device being used to view it. RWD is one approach to the problem of designing for the multitude of devices available to customers, ranging from tiny phones to huge desktop monitors.

The EQRS application is built using the Bootstrap grid system which provides a flexible, device-agnostic layout system.

Responsive Breakpoints

Breakpoints are the point at which your sites content will respond to provide the user with the best possible layout to consume the information. They are the media query values that will mark the transition to a new class of devices.

Media query usage:

  • We use breakpoints for tablets and desktop devices.
  • Our defined media queries start from 768px to 991px for tablet devices in portrait mode.
  • Desktop layout starts at 992px and up.

Tips

  • When designing for specific breakpoints, design for the amount of screen space actually available to your site
  • Use Fluid Images.
  • Pay Attention to Font Sizes.
  • Test the application using a real device.
  • Avoid the use of large graphics.
