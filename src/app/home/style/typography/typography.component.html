Typography

EQRS uses two font families. Merriweather, $eqrs-serif-font-family , is used for most of the headers. Source Sans Pro, $eqrs-sans-serif-font-family, is used mainly for the body and detail copy because of its legibility at smaller font sizes.

We use rem for all font sizing to allow for greater accessibility. One rem is equal to 18 pixels within EQRS.

Ensure headings are in a logical order. For example, check that all headings are marked with <h1>-<h2> elements and that these are ordered hierarchically. For example, the heading level following an <h1> should be an <h2>, not an <h3>. Heading elements must be properly nested on the page. Heading elements that are properly nested help users of assistive technologies understand the structure of the information on the web page. Finally, don't use heading mark up on text that isn't actually a heading.

Display

Use CSS class eqrs-display-text, defined as:

  • font-family: ‘Merriweather’
  • font-weight: 400
  • font-size: 52px
  • line-height: 4.25em/68px

Heading 1

CSS for h1 defined as:

  • font-family: ‘Merriweather’
  • font-weight: 700
  • font-size: 40px
  • line-height: 3.25em/52px

Heading 2

CSS for h2 defined as:

  • font-family: ‘Merriweather’
  • font-weight: 700
  • font-size: 30px
  • line-height: 2.44em/39px

Heading 3

CSS for h3 defined as:

  • font-family: ‘Merriweather’
  • font-weight: 700
  • font-size: 22px
  • line-height: 1.63em/26px

Heading 4

CSS for h4 defined as:

  • font-family: ‘Source Sans Pro’
  • font-weight: 700
  • font-size: 20px
  • line-height: 1.38em/22px
  • text-transform: uppercase;

Body copy

CSS for p defined as:

  • font-family: ‘Source Sans Pro’
  • font-weight: 400
  • font-size: 18px/1rem
  • line-height: 1.63em/26px

Detail copy

Use CSS class font-sans-3xs defined as:

  • font-family: ‘Source Sans Pro’
  • font-weight: 300
  • font-size: 14px
  • line-height: 1.38em/22px

Alignment

In most cases, align content to the left of its respective container.

Example

I am a title.

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nunc quis lacinia enim, vitae commodo leo. Quisque ut lobortis augue. Praesent ac diam dignissim, accumsan odio nec, commodo metus.

Line length

Keep line length to approximately 66 characters per line (max-width of 35em) and allow for some white space on the page. Anything more can cause eye strain, when reading multiple lines of text.

Capitalization

Use sentence case for all navigation, headers, labels and buttons.

Emphasizing text

For emphasis within text use em or strong tags to change the voice of a word or phrase, i.e., to emphasize it. Otherwise, use b tags to change the visual look and feel only. Avoid using italics because it can be difficult to read and to decipher meaning.
