Date

Date Pipe is used to format isoDateStringNoTime 'yyyy-MM-dd' to default 'MM/DD/YYYY' or a specified format.

[required]="true"
will trigger required validation and will throw 'required' error

[minValue]="'1973-01-01'"
will trigger minimum date validation if the user provided date is before '1973-01-01' and will throw 'minDate' error

[maxValue]="'2010-01-01'"
will trigger maximum date validation if the user provided date is after '2000-01-01' and will throw 'maxDate' error

[setErrorStyle]="form.controls['dateControl'].touched && form.controls['dateControl'].invalid"
The setErroStyle attribute accepts the condition when the error styling needs to be applied.

[id]="'myControl'"
will set element id's to myControl_month, myControl_day and myControl_year respectively.

[label]="'Effective Date'"
will update the aria-label for month input tag as 'Effective Date Month'
will update the aria-label for day input tag as 'Effective Date Day'
will update the aria-label for year input tag as 'Effective Date Year'
